for commit/uploading changes.
1) to see all the files that were changed, only names not content.
git diff --name-only
git status

2) file name will be generated by git diff --name-only, use that to,add files that you would like to commit,

git add <file name>

3) Type a commit message
git commit,

for fetching changes
IMPORTANT: if you have any local changes, they will be lost. With or without --hard,
any local commits that haven't been pushed will be lost.

git fetch --all
git reset --hard origin/master
